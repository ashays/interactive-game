AP Computer Science / Question Sets /Data Structures Overview

  • Array A fixed length list of items that can be accessed by index, added to or removed from
  • List A dynamic length list of items that can be accessed by index, added to or removed from
  • Tuple An immutable list of items that can be accessed by index but not modified in any way
  • Stack Last-in-first-out (LIFO) data structure, push adds to top, pop removes from top
  • Queue First-in-first-out (FIFO) data structure, push adds to back, pop removes from front
